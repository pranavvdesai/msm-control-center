# MSM Control Center

**Private attendance intelligence for TAPMI MSM cohort**

Developed by **Raam** — Naam toh suna hoga !!

## Live Features

- Cinematic superhero login & welcome animation
- Smart leave tracking (3/2/1 credit = 3/2/1 leaves)
- Calendar → lecture → Regular / Condoned leave flow
- Attendance risk meter + gamified Bollywood meme alerts
- Live class social feed
- Class leaderboard
- Excel timetable upload (TERM 4 MBA-MKT TT.xlsx)
- Mobile PWA with bottom navigation
- Supabase PostgreSQL production ready

## Quick Start (Local)

```bash
npm install
npm run db:setup:local
npm run dev
```

**http://localhost:3000** · `raam@msm.cohort` / `msm2026`
